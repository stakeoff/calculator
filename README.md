# Calculadora Simples em Python

Uma calculadora bem básica e direta feita em Python 3 – ideal pra quem tá começando a programar!

### Operações disponíveis
- `+` soma  
- `−` subtração  
- `×` multiplicação  
- `/` divisão  

### Como usar
1. Baixe ou clone o repositório  
2. Abra o terminal na pasta do projeto  
3. Rode o comando:
   ```bash
   python calculadora.py
